## 実行環境

- Ubuntu on WSL2
- fish

SSH キーを追加し，GitHub に公開鍵を登録しましょう．
まず，鍵を生成します.

```sh
ssh-keygen -t ed25519 -C "my_email@domain.ninja"
```

公開鍵認証で使用するための認証鍵の情報を保存するためにエージェントを起動します．

```sh
eval (ssh-agent -c)
```

作成した秘密鍵を登録します．

```sh
ssh-add ~/.ssh/my-ssh-key
```

GitHub への登録は[Settings](https://github.com/settings/profile)から可能です．
